# Web UI Prototype

Минималистичный прототип веб-интерфейса для:

- лендинга;
- авторизации с восстановлением сессии по cookie;
- рабочего экрана с чатом, загрузкой файла и историей;
- отдельной страницы истории запросов.

Сервис авторизации, `сервис а` и `сервис б` пока замоканы внутри приложения. Точки замены вынесены в `app/services/`.

## Быстрый запуск

```bash
python -m venv .venv
.venv\Scripts\activate
pip install -r requirements.txt
uvicorn app.main:app --reload
```

После запуска:

- лендинг: `http://127.0.0.1:8000/`
- логин: `http://127.0.0.1:8000/login`
- рабочая область: `http://127.0.0.1:8000/workspace`
- история: `http://127.0.0.1:8000/history`

## Что можно менять позже

- `app/services/auth.py`: подключение реального сервиса авторизации и JWT;
- `app/services/chat.py`: отправка файла и prompt в `сервис а`, ожидание результата от `сервиса б`;
- `app/config.py`: базовые URL, имена cookie и TTL.

Сейчас логин принимает любые непустые `username` и `password`.
